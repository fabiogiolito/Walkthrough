Code Walkthrough

Add comments to your code blocks with this plugin for Prism.js

Made by Fabio Giolito. Code available on Github.

Markup:

    
      <pre 
        class="language-markup line-numbers" 
        data-comments='[
          {"3-8":"A comment highlighting lines 3 to 8."},
          {"4":"Another comment, highlighting line 4."},
          {"":"A comment without line highlighting"},
          …
        ]'
      >
        <code>…</code>
      </pre>

Instalation:

    
      <script src="prism/prism.js"></script>
      <link rel="stylesheet" href="prism/prism.css">
  
      <script src="plugin/walkthrough.js"></script>
      <link rel="stylesheet" href="plugin/walkthrough.css">

Customizing:

    
      pre.custom-walkthrough .line-highlight {
        background: #4D3A4D;
      }

      pre.custom-walkthrough {
        width: 640px;

      }

        position: absolute;
        top: 0;
        right: calc(-260px - 3em);
        margin: 0;
        background: #F0DDF0;
        color: black;
        flex-direction: column-reverse;
      }
